# 💧 Simulador Go-to-Market: AquaLevel Perú

Este es un simulador interactivo diseñado para estudiantes de **Administración y Marketing**. El objetivo es configurar una estrategia de lanzamiento para una marca de agua de mesa gamificada en el mercado peruano.

## 🚀 Cómo ejecutarlo localmente
1. Clona este repositorio.
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta la app: `streamlit run app.py`

## 🕹️ Dinámica del Caso
El estudiante debe responder las 6 preguntas del **Go-to-Market (GTM)**:
- ¿Quién?
- ¿Qué?
- ¿Por qué?
- ¿Dónde?
- ¿Cómo?
- ¿Cuándo?

El sistema evalúa la coherencia estratégica y entrega métricas clave: **CAC, CLTV y ROI en Soles**.
